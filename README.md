# 🏳️ Vexicon

Vexicon is a word game that transforms country names and nationalities into flag emojis. It combines elements of language, geography, and puzzles into a simple interactive experience.

## Overview

Vexicon lets you encode and decode sentences using country flags. It’s designed for both casual play and daily challenges.

Features include:

- **Encode Mode**: Convert country names and demonyms into flag emojis  
  _Example_: “I love French food and Brazilian music” → “I love 🇫🇷 food and 🇧🇷 music”

- **Daily Challenge**: A new puzzle each day with increasing difficulty (Easy → Medium → Hard)

- **Practice Mode**: Unlimited puzzles to help you improve recognition and recall

## Tech Stack

- React 19  
- TypeScript  
- Vite  
- @faker-js/faker (for generating practice content)

## Getting Started

Clone the repository:

```bash
git clone <your-repo-url>
cd vexicon
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Then open your browser to begin using the app.

## Usage

### Encode Mode
Enter a sentence and the app will replace recognized country names and demonyms with their corresponding flags.

### Daily Challenge
Complete a new set of puzzles each day, progressing through increasing levels of difficulty.

### Practice Mode
Play unlimited decoding challenges. Easier levels use multiple choice, while harder levels require free-text answers.
